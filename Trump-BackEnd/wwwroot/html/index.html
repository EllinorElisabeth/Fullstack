Trump API Documentation

Overview

The Trump API allows users to interact with Trump's thoughts.
You can retrieve, create, update, and delete thoughts.
Additionally, it support image uploads.

REST

Base url: localhost:5115/Trump

Endpoints

GET/Trump

Retrieve all Trump thoughts.

  • Request:

    GET localhost:5115/Trump

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "America Is Great!"
            }
        ]

GET/Trump{id}

Retrieve Trump thought by it's ID.

  • Request:

    GET localhost:5115/Trump/{id}

  • Parameters:

    id (integer)

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "America Is Great!"
            }
        ]

GET/Trump/GetByThought/{thought}

Retrieve Trump thoughts by specific text.

  • Request:

    GET localhost:5115/Trump//GetByThought/{thought}

  • Parameters:

    thought (string)

  • Response:

    200 OK

     
        [
            {
                "id": 1,
                "thought": "America Is Great!"
            }
        ]

POST/Trump

Create a new Trump thought.

  • Request:

    POST localhost:5115/Trump/

  • Request body: 
    {"thought": "China"}
  • Response:

    • 200 OK

    {"id":1, "thought": "China"}

PUT/Trump

Update an exsisting Trump thought.

  • Request:

    PUT localhost:5115/Trump

  • Request body: 
      {"thought":"Let's Make America Even Greater!"}
  • Response:

    200 OK

DELETE/Trump/{id}

Delete Trump thought.

  • Request:

    DELETE localhost:5115/Trump{id}

  • Parameters:

    id (integer)

  • Response:

    200 OK

POST/UploadingImage

Upload an image.

  • Request:

    POST localhost:5115/UploadImage

  • Parameters:

    A file (image)

  • Response:

    200 OK
